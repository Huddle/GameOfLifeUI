Game of life UI

Conway's Game of Life is a zero-player game, a cellular automation. Refresh the page to see a new simulation.

  1. In the first empty box provided, create a completely new visual representation for the existing simulation using a <div> tag.
  2. Now mirror the new visualisation (if the top-right cell of the old visualisation is alive, so is the top-left of the new visualisation).
  1. Now create a new simulation independent of the above games, using your newly created visualisation.
