Game of life UI

Conway's Game of Life is a zero-player game, a cellular automation. This implementation does not allow direct configuration, refresh the page to see a new simulation.

In the first empty box provided, create a new visual representation for the existing simulation using CSS. It should provide a mirrored image of the existing visualisation (if the top-right cell is alive, so is the top-left of the new visualisation). Feel free to change the JavaScript and CSS as needed.

Now create a new simulation independent of the above games, using an existing visual representation. The visualisations should work in all modern browsers.
