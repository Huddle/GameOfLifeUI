Game of life UI

Conway's Game of Life is a zero-player game, a cellular automation. This implementation does not allow direct configuration, refresh the page to see a new simulation. Please clone or download the repo before starting.

In the first empty box provided, create a completely new visual representation for the existing simulation using HTML and CSS. It should provide a mirrored image of the existing visualisation (if the top-right cell is alive, so is the top-left of the new visualisation). You will need to understand and modify the existing JavaScript and CSS.

Now create a new simulation independent of the above games, using your newly created visualisation. The visualisations should work in the latest version of Chrome.
